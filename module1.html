Getting Started w/ HTML & CSS

  1. Add in !DOCTYPE and html tags.
  2. Start with a title and link the file to your css.
  3. Initialize a background, color, and font-family.
  4. Align the text to wherever is most suited.
  5. Start working!
Useful Tags Functionality
link/script You can use this as a directory to your css/js file.
section You can use this as a way to access different areas of your website.
ol/ul/li You can use this to create an ordered or unordered list of items.
table/thead/tr/th You can use this to create a table of items.
